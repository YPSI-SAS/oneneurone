# OneNeurone

Plugin IA pour [ZeroNeurone](https://github.com/YPSI-SAS/ZeroNeurone).

> *Le Neurone propose, l'analyste dispose.*

## Installation

1. Copier `oneneurone.js` dans le dossier `plugins/` de ZeroNeurone
2. Creer ou mettre a jour `plugins/manifest.json` :

```json
{
  "plugins": [
    {
      "id": "oneneurone",
      "file": "oneneurone.js",
      "name": "OneNeurone",
      "description": "Le Neurone propose, l'analyste dispose"
    }
  ]
}
```

3. Relancer ZeroNeurone
